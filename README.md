# Project Name: System Monitoring Dashboard 

# Overview
Essentially a display that showcases the computers real time CPU, memory, disk, and process information in a structured manner. This allows users to gain an insight into their computer's performance by monitoring key system resources.

## Planned Features

- Display CPU, memory, and disk usage
- Display CPU level with corresponding status
- Color coded CPU status
- Auto Refresh every 2 seconds
-Using Python, psutil, and colorama
